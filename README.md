### 🛠 Swiftly Workspace

A multi-platform ERP/CRM built entirely with SwiftUI, targeting **iOS**, **macOS**, and **tvOS**.

- **Frontend** → SwiftUI across all Apple platforms
- **Backend** → Vapor (Swift) deployed on Heroku with PostgreSQL
- **Subscriptions** → RevenueCat + Stripe integration
- **Architecture** → Shared framework with separate client targets per platform
- **Distribution** → Available on the App Store & Mac App Store via TestFlight
